FlappyBird-NEAT

Implementação de uma rede neural artificial utilizando Python com a biblioteca NEAT para aprendizado do jogo Flappy Bird.

Para executar o projeto é necessário utilizar o arquivo `flappy_bird_tutorial.py`.
